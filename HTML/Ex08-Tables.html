Tables in HTML

Tables are ways to represent data in a tabular format where the data will have rows and columns. U create a table using table tag and have in it table rows(tr) that contains one or more tabledata(td).U can create headings to the columns of the table using th instead of td. However, its good practise to create sections within the table as thead, tbody and tfoot.
