What are Forms?

Forms Understanding

Forms are a way of taking inputs from the user and do some processing at the server side. form will have attributes like action and method. action tells about the server side page that will process the inputs given in the form. method tells about the way the inputs are transfered to the action page. There are 4 major ways of transfering the data to the action page.

  • GET : Used to get the information based on inputs U give. Data is shared thru URL
  • POST : Used to send info(Adding) to the server page.
  • PUT : Used to update an existing data to the server page
  • DELETE: Used to delete existing data in the server
