Using Picture Tag

For multiple devices, we use picture tag that takes the attribute called media to difine the kind of the device on which the image is to be displayed. If U want UR Images to be modified for different kinds of media conditions like simplier version of the image for lower bandwidths. It is also used to offer alternate image formats that are not supported on the device. It is also used to save the bandwidths for high resolution images, thereby speeding up the loading time of the image
